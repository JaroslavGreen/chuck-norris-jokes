Chuck Norris Jokes App is a simple web project that fetches random jokes from the Chuck Norris API.
It uses fetch with async/await, includes loading and error states, and provides a clean, responsive UI with a “New joke” button.

Tech stack: HTML, CSS, JavaScript (Fetch API)

Features:

Random joke fetching from public API

Loading + error handling (try/catch, response.ok)

Responsive layout with basic UX improvements﻿# chuck-norris-jokes
